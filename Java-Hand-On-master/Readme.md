# 💻 Java - Hands-On Practice Sessions

This sub-repository contains structured Java source code and solutions for the **TCS Xplore Course Recordings**, **Java Assessments**, and official learning track evaluations.

---

## 📂 Practice Track Modules

The folder contains 9 structured learning directories:

### 🔢 1. Core Basics & Numeric Computations
* **Directory**: [`1. Numeric Computation - Hands on 1`](./1.%20Numeric%20Computation%20-%20Hands%20on%201)
* **Concepts**: Arithmetic operators, typecasting, expressions, parsing numeric strings, and order of operations.

### 🏗️ 2. Classes & Object-Oriented Design
* **Directory**: [`2. Classes and Objects - Hands on 1`](./2.%20Classes%20and%20Objects%20-%20Hands%20on%201)
* **Concepts**: Class structures, object instantiations, attribute declarations, local vs member variables, and object memory maps.

### 🔀 3. Conditional Decision Making
* **Directory**: [`3. Conditional Operation - Hand on 1`](./3.%20Conditional%20Operation%20-%20Hand%20on%201)
* **Concepts**: Single-tier and nested `if-else` blocks, boolean flags, short-circuit operators, and `switch-case` statements.

### 🔁 4. Loops & Iterative Operations
* **Directories**:
  * [`4. Java Iterations - Hands on 1`](./4.%20Java%20Iterations%20-%20Hands%20on%201)
  * [`5. Java Iterations - Hands on 2`](./5.%20Java%20Iterations%20-%20Hands%20on%202)
  * [`6. Java Iterations - Hands on 3`](./6.%20Java%20Iterations%20-%20Hands%20on%203)
* **Concepts**: Iterating with `for`, `while`, and `do-while` loops, break/continue directives, nested loop matrices, bounds testing, and recursion basics.

### 📊 5. Array Allocation & Storage
* **Directory**: [`7. Java Arrays - Hands on 1`](./7.%20Java%20Arrays%20-%20Hands%20on%201)
* **Concepts**: Initializing single-dimensional and multi-dimensional arrays, index manipulation, traversing array collections, and array bounds safety.

## 🛠️ Compilation Guide
All source scripts can be compiled and run with any standard Java compiler. For example:
```bash
# Navigate to the target module directory
cd "1. Numeric Computation - Hands on 1"

# Compile all source files
javac *.java

# Execute the driver program class
java Solution
```
Ensure your input formats line up with the prompts. Use an IDE such as VS Code or Eclipse for visual execution and debugging.

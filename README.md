## 🛠️ How to Compile and Run Code Locally

### 1. Compile & Run Java Files
Ensure you have Java JDK (version 8 or higher) installed. 

**Using Terminal/PowerShell:**
```bash
# Navigate to the file location, e.g. the 15 Marks source folder
cd "Important-Java-Program--IPA-15Marks-master/src"

# Compile the target file
javac Anagram.java

# Execute the compiled class
java Anagram
```

**Using an IDE (Recommended):**
Open the parent folder in **VS Code**, **IntelliJ IDEA**, or **Eclipse**. Set up the project source directories and run directly from the IDE interface.

### 2. SQL & PL/SQL Practice
You can run the SQL snippets found in the MCQ guide against any local database workspace (like Oracle Express Edition, MySQL, or PostgreSQL) or online SQL compilers:

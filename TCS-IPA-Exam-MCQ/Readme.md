- [SQL PLSQL.md](./SQL%20PLSQL.md): SQL and PL/SQL MCQs, syntax, and important concepts.
- [UI - HTML, CSS AND JAVASCRIPT.md](./UI%20-%20HTML,%20CSS%20AND%20JAVASCRIPT.md): MCQs and coding questions on HTML, CSS, and JavaScript.
- [Know Your Organization.md](./Know%20Your%20Organization.md): Organization-related MCQs and facts.
- [Previous Year MCQ.md](./Previous%20Year%20MCQ.md): Links to previous year MCQ papers and solutions.

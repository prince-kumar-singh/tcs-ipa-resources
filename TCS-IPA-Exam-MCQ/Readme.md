## 📝 Study Tip for Theory Sections
During the exam, questions on Java memory, SQL commands, and CSS properties often have subtle options. 
> [!IMPORTANT]
> - Always double-check index values (e.g. JavaScript substring `substring(startIndex, endIndex)` is exclusive of the end index).
> - Make sure to understand access specifiers (e.g. private fields are only visible within their declaring class).
> - Be clear on Unix file permissions (e.g. `chmod` actions).
